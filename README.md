## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deployment

### Railway Deployment

Deploy this NestJS application is using Railway:

1. Create a Railway account at [railway.app](https://railway.app)
2. Install the Railway CLI:
   ```bash
   npm i -g @railway/cli
   ```
3. Login to Railway:
   ```bash
   railway login
   ```
4. Create a new project:
   ```bash
   railway init
   ```
5. Add a PostgreSQL database:
   ```bash
   railway add
   ```
   Select PostgreSQL when prompted.
6. Deploy the application:
   ```bash
   railway up
   ```
7. Set environment variables:
   ```bash
   railway vars set DATABASE_URL=YOUR_POSTGRESQL_URL
   railway vars set CLERK_SECRET_KEY=YOUR_CLERK_SECRET_KEY
   railway vars set CLERK_API_URL=https://api.clerk.com/v1
   railway vars set SENDGRID_API_KEY=YOUR_SENDGRID_API_KEY
   railway vars set SENDGRID_EMAIL_FROM=YOUR_SENDER_EMAIL
   ```
8. Open your application:
   ```bash
   railway open
   ```
